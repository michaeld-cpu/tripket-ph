# Tripket PH — Admin Dashboard

Next.js 14 (App Router) + TypeScript + Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

## Pages
- `/` Dashboard — KPIs, revenue chart, today's voyages
- `/vessels` Fleet management
- `/routes` Route catalogue
- `/voyages` Scheduled voyages with load %
- `/bookings` Passenger bookings + revenue summary

## Mock API
- `GET /api/vessels`
- `GET /api/routes`
- `GET /api/voyages`
- `GET /api/bookings`
